Test Case Requirements

Test Case: TCD-002

Summary:
Invalid Login
Preconditions:
None
Test Steps:
Step Description
1 Input valid/invalid User Name
2 Input valid/invalid Password
3 Login
Expected Results:
User should not be able to login, an error page is opened
Notes:

Test the combinations for:

  1. Invalid User Name and Invalid Password
  2. Invalid User Name but Valid Password
  3. Valid User Name but Invalid Password
